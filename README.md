🏥 Medicore Website

A responsive healthcare website built using HTML, CSS, and JavaScript. This project showcases a modern medical platform with features like doctor listings, appointment booking UI, notifications, and interactive components.

🚀 Features
🩺 Doctor listing and details
📅 Appointment booking interface
🔔 Notification system
🔍 Search functionality
📱 Fully responsive design
🎨 Clean and modern UI
⚡ Smooth animations and interactions
🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Interactivity and dynamic features
